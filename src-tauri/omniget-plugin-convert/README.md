# OmniGet Media Converter Plugin

Convert videos and audio between formats.

## Features

- Convert between video formats (MP4, MKV, WebM, AVI, MOV)
- Extract audio (MP3, AAC, FLAC, WAV, OGG)
- GPU hardware acceleration (NVIDIA NVENC, AMD AMF, Intel QSV)
- Probe media files for format/codec info
- Real-time progress with cancel support

## Install

This plugin is bundled with OmniGet. It can also be installed from the in-app Marketplace.

## Development

```bash
cargo check -p omniget-plugin-convert
```
